Udemy Clone 🎓
A responsive Udemy-inspired landing page built using HTML5 and CSS3. This project recreates the basic layout and design of the Udemy website, including the navigation bar, categories, featured courses, and footer section.

🚀 Features
Responsive navigation bar
Hero/banner section
Course categories section
Featured courses display
Recommended courses section
Topics section
Footer with useful links
Clean and modern UI design
Fully responsive layout using CSS
🛠️ Technologies Used
HTML5
CSS3
Google Fonts
Font Awesome (Optional)
